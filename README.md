# Neck Analysis

首の動き・安定性・形態的特徴を計測し、参考指標を算出するWebアプリ

## 概要

カメラで首の動き（右回旋・左回旋・上向き）を計測し、
複数の身体的指標を組み合わせて参考スコアを算出します。

## 技術スタック

- React 18 + Vite
- MediaPipe Face Mesh（顔ランドマーク検出）
- Chart.js（レーダーチャート）

## セットアップ

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

`dist/` フォルダにビルド成果物が生成されます。

## 注意事項

- **医療目的ではありません**
- **能力評価ではありません**
- **エンターテインメント目的の参考指標です**

算出されるスコアや判定は、あくまで参考値であり、
実際の身体能力や競技適性を保証するものではありません。
